# [F01] 반복 유형 선택 (Repeat Type Selection)

- 🎯 목적(Goal)

  - 일정 생성/수정 시 반복 유형을 선택할 수 있도록 한다.

- 📘 기능 설명(Feature Description)

  - 사용자는 반복 유형으로 매일, 매주, 매월, 매년 중 하나를 선택할 수 있다.
  - 월 31일 규칙: 31일에 "매월" 선택 시 31일에만 생성한다.
  - 윤년 규칙: 2월 29일에 "매년" 선택 시 2월 29일에만 생성한다.

- ⚙️ 요구사항(Requirements)

  - [ ] 반복 유형 옵션: 매일, 매주, 매월, 매년 제공
  - [ ] "매월" 선택 시 월 31일 규칙을 적용
  - [ ] "매년" 선택 시 윤년 2/29 규칙을 적용
  - [ ] 반복일정 생성 시 일정 겹침은 고려하지 않음

- 🧩 예외/제한사항(Constraints)

  - 반복일정은 일정 겹침을 고려하지 않는다.
  - 시작일/종료일의 타임존 기준은 로컬 타임존을 따른다.

- ✅ 완료 기준(DoD)

  - [ ] 각 반복 유형별로 인스턴스 생성 규칙에 대한 단위 테스트 존재
  - [ ] 31일/윤년 규칙에 대한 경계값 테스트 존재
  - [ ] 월/주/일 뷰에서 반복 인스턴스가 기대대로 보임(통합 테스트)

- 🧠 우선순위(Priority)
  - High
